### Code example
**Summation function KATA from CODEWARS** *Write a program that finds the summation of every number from 1 to num. The number will always be a positive integer greater than 0.*
```
var summation = function (num) {
  let i = 0;
  let y = 1;
  let sum = i + y;
  for(let y = 2; y <= num; y++){
    sum += y;
  }
  return sum;
}
```
